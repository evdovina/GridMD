gmThreadPool::~gmThreadPool ( )

Destructor of the Thread Pool.

Kills all working and waiting threads and destroys all queued and processed tasks. Destructor blocks caller thread until all the threads are destroyed.

void gmThreadPool::RemoveTask ( gmTaskID  taskID)

Removes task with taskID index from pool queue and destroys it.

After call to this function gmThreadPool::IsValidIndex() for queried taskID returns false. If task is being processed, it will be killed immediately. If taskID index is not valid (see gmThreadPool::IsValidIndex()), does nothing.

int gmThreadPool::TaskResult ( gmTaskID  taskID)

Returns task exit code.

Waits for task with taskID index to complete – blocks the caller thread until queried task is performed. After destroys task object and returns task exit code. If taskID index is not valid (see gmThreadPool::IsValidIndex()) returns GMPOOLTASK_INVALID_RESULT (see definition in gmthreaddefs.h).

gmTASK_STATUS gmThreadPool::TaskStatus ( gmTaskID  taskID)

Gets task current status.

Possible return values are:
gmTASK_POOLED – task has been queued for processing.
gmTASK_PROCESSED – task is being processed by any of the worker threads.
gmTASK_FINISHES – task succesfully finished. call gmThreadPool::TaskResult() to get its exit code and destroy it.
gmTASK_INVALID_STATUS – is returned if taskID index is not valid (see gmThreadPool::IsValidIndex()).
