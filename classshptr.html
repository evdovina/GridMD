Detailed Description

template<class T, class delete_t = delete_ptr<T>>
class shptr< T, delete_t >

Smart pointer.

Used to control memory allocated for dynamic objects. Contains pointer p on dynamic object and pointer on integer number num, which specify the number of other smart pointers with refers at the same dynamic object p. If *num becomes zero (no one refers to dynamic object), dynamic object should be deleted. Умный указатель, содержащий в себе указатель на динамический объект (p) и указатель на счетчик ссылок на него (num) При нулевом количестве ссылок, указывающих на динамический объект, динамический объект удаляется
