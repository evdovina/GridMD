GridMD

Overview

GridMD is the C++ class library intended to help developers to quickly build molecular dynamics or Monte-Carlo simulation programs and to run them in the distributed environment.

GridMD library is free software. The full licence is available in the file /gridmd/doc/gridmd_license.txt

Home project web page: http://gridmd.sourceforge.net/

Doxygen documentation page: http://imorozzov.github.io/GridMD/

How to build

Dependencies

  • Boost headers
  • Either
    • Boost libraries filesystem, regex, system, timer (default) OR
      • WxWidgets base library*
  • libssh (default) or libssh2
  • libxml2 with iconv (optional, enabled by default)
